### ⚙️ Tech Stack
The bulk of the code editing is done in VSCodium and GNOME Text Editor (based on gedit). Nowadays I exclusively work in C++, GDScript and JavaScript. I have some knowledge of Kotlin, Java, C#, but I have no plans to work again in these languages at this time.

For game development, I use Godot Engine 4 (using the latest commit on the `main` branch), using my Sapphire plugin toolkit. I use GDScript, I used to use C++ when I had my own fork and used modules, I still use C++ in GDExtensions, when I need to use them.

I used Unreal Engine versions 4.23 through 5.3 from 2019, but [I dropped it entirely at the end of 2024](https://alextecplayz.com/post/roadmap/2024/01/13/Sapphire-thread.html) when I used my Godot fork, and then transitioned to just Godot 4.

I (*very* rarely) use Android Studio to occasionally test out new features of the Android API, but I am not working on developing native Android apps anymore.

For my (occasional) art, I use MagicaVoxel (for voxel-based art), Blender (for 3D art), and Aseprite (for 2D art).
